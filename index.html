Help

The application calculates the survivability and equilibrium values at each point of the game.
The buttons immediately below the chessboard allow you to move the game back to the starting point, move back to the previous move, move forward to the next move and move forward to the last move.
The "Upload Game" button allows you to introduce into the system a file with .pgn format that you want to analyse.
The "Additional Information" button allows you to view additional game data.
The "Download Survivability and Equilibrium Values" button allows you to download a database with the survivability and equilibrium values data of the uploaded game. and survivability values of the loaded game.
In each of the tables you will find the last ten survival values with respect to the current movement of the different pieces.
The table on the right shows the moves of the game.
The buttons below the move table allow you to move to the next ten moves or to the previous ten moves.
